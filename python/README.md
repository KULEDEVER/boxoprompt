# SCP Researcher

Run:

```powershell
python app.py
```

Features:

- PyQt6 Foundation internal utility interface
- SCP lookup with browser opening
- Searchable SCP index with object-class filters
- Center file preview panel
- Favorites
- Access history
- Random SCP selector
- Local profile storage in `%LOCALAPPDATA%\SCP Researcher\profile.json`
- Amber, Red, and Blue QSS theme modes
