Platform notes

What to know before you install

The app is built with Java Swing and standard Java libraries. Install Java 8 or newer first, then run the JAR like a normal desktop utility.

Compatibility This release is intended for systems that can run a modern Java runtime.
Development status The app is still being improved, so feedback is welcome while the full GUI utility suite grows.
